🤖 민원처리 챗봇 시스템

✅ 시스템 준비 완료!

모든 파일이 정상적으로 로드되었습니다. 아래 버튼을 클릭하여 챗봇을 시작하세요.

📁 프로젝트 파일 목록

📄 index.html - 메인 챗봇 페이지
🎨 styles.css - 파랑~보라 로봇 테마 스타일
⚙️ script.js - 챗봇 기능 및 상호작용
🚀 챗봇 시작하기

💡 사용 방법

  1. 위의 "챗봇 시작하기" 버튼을 클릭하세요
  2. 민원 예시 버튼을 클릭하거나 직접 입력하세요
  3. 봇과 대화한 후 "답변생성" 버튼을 클릭하세요
  4. 답변추천과 관련법령을 확인하세요
