# 내부 직원용 민원처리 챗봇

내부 직원이 민원을 받았을 때 처리 방법을 문의할 수 있는 웹 기반 챗봇 시스템입니다.

## 🎯 주요 기능

- 🤖 내부 직원 전용 민원처리 상담 챗봇
- 💬 직관적인 채팅 인터페이스
- ⚡ 빠른 응답 시스템
- 📱 반응형 디자인 (모바일 지원)
- 🎯 빠른 액션 버튼 (자주 묻는 질문)
- 🌐 순수 HTML/CSS/JavaScript (서버 불필요)

## 🚀 실행 방법

### 간단한 실행
1. **`index.html` 파일을 더블클릭**하거나
2. **브라우저에서 파일을 열기**하면 바로 사용 가능

### 로컬 서버로 실행 (선택사항)
```bash
# Python 3가 설치된 경우
python -m http.server 8000

# 또는 Node.js가 설치된 경우
npx http-server
```

## 📋 지원하는 질문 유형

### 🎯 빠른 액션 버튼
- **처리 방법 문의**: 민원 처리 절차 안내
- **법령/규정 문의**: 관련 법령 및 규정 정보
- **유사 사례 검색**: 과거 유사 민원 사례
- **보고 필요성**: 상급자 보고가 필요한 경우
- **처리 기간**: 민원 처리 소요 기간
- **답변 내용**: 민원인 답변 가이드

### 🔍 키워드 기반 응답
- **처리**: 민원 처리 절차 및 방법
- **법령**: 관련 법령 및 규정
- **사례**: 유사 민원 사례 검색
- **보고**: 상급자 보고 기준
- **기간**: 처리 소요 기간
- **답변**: 민원인 답변 방법
- **긴급**: 긴급 민원 처리 절차
- **예산**: 예산 소요 민원 처리
- **정책**: 정책 변경 요청 민원
- **협조**: 타 기관 협조 필요 민원

## 💼 사용 시나리오

### 시나리오 1: 새로운 민원 접수
```
직원: "도로 보수 요청 민원이 들어왔는데 어떻게 처리해야 하나요?"
챗봇: 민원 처리 절차와 관련 부서 협의 방법 안내
```

### 시나리오 2: 법령 확인
```
직원: "개인정보 관련 민원인데 관련 법령이 있나요?"
챗봇: 개인정보보호법 및 관련 규정 안내
```

### 시나리오 3: 유사 사례 검색
```
직원: "유사한 민원 사례가 있나요?"
챗봇: 과거 유사 민원 처리 사례 검색 안내
```

### 시나리오 4: 상급자 보고
```
직원: "예산이 소요되는 민원인데 보고해야 하나요?"
챗봇: 상급자 보고가 필요한 경우 안내
```

## 🔧 커스터마이징

### 응답 데이터 수정
`index.html` 파일의 `mockResponses` 객체에서 응답 내용을 수정할 수 있습니다:

```javascript
const mockResponses = {
    '처리': '새로운 처리 절차 내용...',
    '법령': '새로운 법령 정보...',
    // 추가 키워드와 응답
};
```

### 빠른 액션 버튼 추가
HTML의 `quick-actions` 섹션에 새로운 버튼을 추가할 수 있습니다:

```html
<button onclick="setQuickMessage('새로운 질문')">새로운 버튼</button>
```
